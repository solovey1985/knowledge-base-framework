# Welcome to {{PROJECT_NAME}}

This is your knowledge base. Start by creating markdown files in the `content` directory.

## Features

- 📝 Markdown support with syntax highlighting
- 🔗 Automatic navigation generation
- 📊 Mermaid diagram support
- 🧮 LaTeX math rendering
- 🔍 Table of contents generation
- 📱 Responsive design

## Getting Started

1. Add your markdown files to the `content` directory
2. Run `npm start` to serve the knowledge base
3. Run `npm run build` to generate a static site

Happy documenting! 🎉
